# sample-spamjs-filters
```
git clone https://github.com/lnt/sample-spamjs-filters.git
cd sample-spamjs-filters
npm install
bower install

grunt build

grunt go

```

## Features

1. Filters by Age slider , Gender, Tags 
2. Sort by feature 
3. Search by keywords
4. Grid vs List view
